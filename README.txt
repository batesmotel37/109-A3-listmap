listmap is program which creates and maintains a hashtable. The user may input new key,value pairs (by entering "key =  value", where key and value are strings), print out the value associated with an existing key (by entering the key in question), or print out the existing hashtable (by entering "=").

When starting the program, input files are also accepted as arguments, their contents is interpretted as commands to populate or print from the hashtable.

The purpose of this excersize is to train a programmer to use templates in C++
